# 🎓 Entity — EdTech Platform & Mobile App

**Entity** — это комплексная кросс-платформенная система (Mobile + Web), разработанная для помощи абитуриентам Казахстана в выборе профессии и расчете вероятности получения образовательного гранта.

> 🔒 **Note:** Исходный код проекта находится в приватном репозитории в связи с NDA и подготовкой к защите дипломного проекта. Данный репозиторий выступает в качестве архитектурной и продуктовой витрины.

---

## 🚀 Ключевой функционал (Core Features)

* 📊 **Многофакторный калькулятор грантов ЕНТ:** Предиктивный алгоритм, рассчитывающий шансы на поступление (Высокий/Средний/Низкий). Учитывает результаты профильных предметов, а также государственные модификаторы (сельская квота, социальные квоты, региональные коэффициенты).
* 🧠 **Профориентационная система (RIASEC):** Интерактивное тестирование из 48 вопросов, вычисляющее код Голланда. Результаты визуализируются через Radar Chart и автоматически сопоставляются с каталогом профессий.
* 📈 **Агрегатор рынка труда (Парсер hh.kz):** Встроенный микросервис и Python-демон для сбора актуальных данных о зарплатах и требуемых навыках. Работает через гибридный подход: OAuth API hh.ru + автоматический fallback на парсинг HTML (Jsoup).
* 🤖 **Контекстно-зависимый ИИ-помощник (Entity AI):** Smart-чат внутри мобильного приложения. Считывает профиль пользователя (баллы, тесты) и генерирует персонализированные советы в формате потоковой передачи данных.

## 🛠 Технологический стек (Tech Stack)

### 📱 Client-side (Mobile & Web)
* **Mobile (Flutter):** Dart, Dio (с интерцепторами для JWT), ValueNotifier (реактивный стейт), flutter_secure_storage, fl_chart, кастомная дизайн-система.
* **Web Frontend (React):** React 19, TypeScript, Vite, Zustand, Tailwind CSS, Framer Motion, i18next (RU, KK, EN).

### ⚙️ Backend & Data Mining
* **Core API:** Java 21, Spring Boot 3.2, RESTful API (OpenAPI/Swagger), Spring Security + JWT.
* **Database:** PostgreSQL, Spring Data JPA, Hibernate, HikariCP.
* **Data Scraper:** Java 17 + WebFlux + Jsoup (асинхронный сбор данных с ротацией прокси).
* **Daemon:** Python 3 + requests + schedule (cron-планировщик обновления базы).
